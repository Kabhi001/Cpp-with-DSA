## 🚀 Getting Started

### Prerequisites

- C++ Compiler (GCC, MinGW, or Visual Studio)
- Basic understanding of programming concepts

### Compilation

To compile any C++ file:

```bash
g++ filename.cpp -o filename
```

To run the compiled program:

```bash
./filename        # On Linux/Mac
filename.exe      # On Windows
```

## 💡 How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/Kabhi001/Cpp-with-DSA.git
   cd Cpp-with-DSA
   ```

2. **Navigate to any topic folder**
   ```bash
   cd Basics
   ```

3. **Compile and run the programs**
   ```bash
   g++ Hello.cpp -o Hello
   ./Hello
   ```

4. **Learn and experiment** - Modify the code, try different inputs, and understand the concepts!
